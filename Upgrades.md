# Технологии для апгрейда

## Байесовская оптимизация

Байесовская оптимизация — это способ поиска оптимальных параметров с минимальным числом попыток.
Optuna или Hyperopt

## Минимальный рабочий стек (рекомендую)

Если кратко, что реально используют:

XGBoost — модель
TimeSeriesSplit / walk-forward вручную — валидация
Optuna — параметры + отбор признаков
Ablation test — финальная проверка признаков
MLflow — это инструмент для управления жизненным циклом ML-моделей.

В связке XGBoost + Optuna + Walk-Forward — практически стандарт.

## Системы мониторинга (infra)

Prometheus + Grafana

Сбор численных метрик.
Дашборды и оповещения.
Используются для latency, error-rates, throughput, custom ML-метрик.
